## Qu'est-ce que SeasonLoot ?

SeasonLoot est un navigateur de tables de butin gratuit pour **World of Warcraft: The War Within — Midnight Saison 1**. Choisissez votre classe, filtrez par les statistiques dont vous avez besoin, et voyez instantanement chaque piece d'equipement disponible cette saison — ou elle tombe, quelles statistiques elle possede et quel est son niveau d'objet.

Pas d'addon a installer. Pas de compte a creer. Ouvrez simplement [seasonloot.com](https://seasonloot.com/fr/) et commencez a planifier votre equipement.

## Les 13 classes

Chaque classe WoW dispose d'une page de butin dediee affichant la liste complete d'equipement de la saison en cours :

[Guerrier](https://seasonloot.com/fr/warrior/) · [Paladin](https://seasonloot.com/fr/paladin/) · [Chasseur](https://seasonloot.com/fr/hunter/) · [Voleur](https://seasonloot.com/fr/rogue/) · [Prêtre](https://seasonloot.com/fr/priest/) · [Chevalier de la mort](https://seasonloot.com/fr/deathknight/) · [Chaman](https://seasonloot.com/fr/shaman/) · [Mage](https://seasonloot.com/fr/mage/) · [Démoniste](https://seasonloot.com/fr/warlock/) · [Moine](https://seasonloot.com/fr/monk/) · [Druide](https://seasonloot.com/fr/druid/) · [Chasseur de démons](https://seasonloot.com/fr/demonhunter/) · [Évocateur](https://seasonloot.com/fr/evoker/)

Chaque page liste **chaque objet equipable** pour cette classe, organise par emplacement ou par source. Vous pouvez affiner par specialisation — par exemple, Maitre brasseur / Marche-vent / Tisse-brume pour le Moine.

## Ce que vous pouvez consulter

### Tous les emplacements d'equipement

Tete, Cou, Epaule, Cape, Torse, Poignets, Mains, Taille, Jambes, Pieds, Anneau, Bijou et Arme — tous les emplacements d'equipement au meme endroit.

### Toutes les sources de butin

Voyez exactement ou chaque objet tombe :

- **8 Donjons** — Siège du triumvirat, Orée-du-Ciel, Cavernes de Mythsara, Tour de Coursevent, Fosse de Saron, Terrasse des Magistères, Nœud Shinas, Académie d'Algeth'ar
- **3 Raids** — Faille onirique, Flèche du Vide, Marche sur Quel'Danas
- **Ensembles de set** — Les 9 pieces de set par classe, incluant les bonus de set 2 pieces et 4 pieces par specialisation

### Ce que chaque objet affiche

Chaque objet sur SeasonLoot affiche :

- **Nom de l'objet** — Nom officiel localise par Blizzard dans votre langue
- **Emplacement d'equipement** — Tete, Torse, Arme, Bijou, etc.
- **Statistiques secondaires avec valeurs exactes** — ex. Crit. 82 / Polyvalence 48
- **Type d'armure** — Tissu, Cuir, Mailles ou Plaques
- **Niveau d'objet** — Drops de donjon (ilvl 246), Drops de raid (ilvl 269–276), Pieces de set (ilvl 263)
- **Source du drop** — Le donjon specifique, le boss de raid ou l'ensemble de set
- **Sous-type d'arme** — Baton, Arme d'hast, Epee a une main, Arme de pugilat, Masse a une main, Hache a une main, Main gauche, etc.
- **Effets de bijou** — Si le bijou est Utiliser (actif) ou Equiper (passif), avec sa statistique principale

## Filtres puissants

Ne faites pas defiler des centaines d'objets. Utilisez les filtres integres pour trouver exactement ce que vous cherchez :

| Filtre | Fonction |
|--------|----------|
| **Specialisation** | Afficher uniquement les objets equipables par votre specialisation (ex. Paladin Sacre, Guerrier Fureur) |
| **Statistiques** | Filtrer par statistique secondaire — Crit., Hâte, Maîtrise ou Polyvalence |
| **Source** | Afficher uniquement le butin de Donjon, de Raid ou les pieces de set |
| **Emplacement** | Aller directement a un emplacement d'equipement specifique |
| **Donjon** | Filtrer les drops d'un donjon specifique |
| **Mode d'affichage** | Basculer entre la vue par emplacement ou par source |
| **Recherche** | Tapez un nom d'objet pour le trouver instantanement |

**Exemple :** Vous jouez un Moine Marche-vent Hâte/Maîtrise ? Selectionnez la specialisation Marche-vent, cochez Hâte et Maîtrise, et voyez instantanement uniquement les objets correspondants — ainsi que le donjon ou le boss de raid exact que vous devez farmer.

## Favoris et partage

- Appuyez sur l'**icone coeur** sur n'importe quel objet pour le sauvegarder dans vos favoris
- Vos favoris persistent entre les visites — stockes localement dans votre navigateur, aucune connexion requise
- Utilisez **"Demander a un ami"** pour generer un lien partageable de votre liste d'equipement et l'envoyer a vos coequipiers de guilde

## Source des donnees et precision

Toutes les donnees sur SeasonLoot sont **collectees directement depuis le client de jeu World of Warcraft** a l'aide d'addons personnalises en jeu. Le processus de collecte lit le Guide de l'aventurier (tables de butin des donjons et raids) et le systeme d'affichage des ensembles de set pour capturer chaque objet, ses statistiques, son niveau d'objet et sa source de drop.

Les noms d'objets dans chaque langue proviennent de l'**API officielle Battle.net de Blizzard**, garantissant que les noms d'objets, les libelles de statistiques et les infobulles correspondent exactement a ce que vous voyez dans le jeu.

Le jeu de donnees actuel couvre l'integralite du pool de butin de Midnight Saison 1 (Patch 4.2.x) :

- **2 000+ objets** sur les 13 classes
- **8 donjons Mythique+ saisonnier** avec les tables de butin completes des boss
- **3 instances de raid** avec tous les drops des boss en difficulte Heroique
- **117 pieces de set** sur 13 classes avec les bonus de set par specialisation
- **Niveaux d'objet precis** — Donjon ilvl 246, Raid ilvl 269–276, Set ilvl 263
- **Valeurs de statistiques secondaires precises** pour chaque objet

Les donnees sont mises a jour chaque fois que Blizzard apporte des modifications aux objets ou qu'une nouvelle saison est lancee.

## A qui s'adresse SeasonLoot ?

- **Joueurs de donjons** — Sachez quelles cles pousser et quels boss cibler pour vos ameliorations
- **Raideurs** — Decouvrez quels boss droppent votre equipement Best in Slot (BiS) avant la soiree raid
- **Joueurs de rerolls** — Voyez rapidement ce qui est disponible pour une classe que vous equipez pour la premiere fois
- **Planificateurs de build** — Filtrez par priorite de statistiques souhaitee pour planifier un set d'equipement optimal
- **Joueurs occasionnels** — Parcourez l'equipement de la saison en cours sans naviguer sur des sites de bases de donnees complexes
- **Joueurs internationaux** — Consultez l'equipement dans votre langue avec les traductions officielles Blizzard
