## SeasonLoot 是什么？

SeasonLoot 是一个免费的魔兽世界装备掉落查询网站，覆盖 **魔兽世界：至暗之夜 赛季1** 的全部掉落数据。选择你的职业，按照你想要的属性进行筛选，就能立刻看到这个赛季所有你能穿的装备 —— 在哪掉落、有什么属性、装等多少。

不用装插件，不用注册账号。打开 [seasonloot.com](https://seasonloot.com/zh-cn/) 就能开始规划你的装备。

## 覆盖全部 13 个职业

每个职业都有专属的掉落页面，展示当前赛季的完整装备列表：

[战士](https://seasonloot.com/zh-cn/warrior/) · [圣骑士](https://seasonloot.com/zh-cn/paladin/) · [猎人](https://seasonloot.com/zh-cn/hunter/) · [盗贼](https://seasonloot.com/zh-cn/rogue/) · [牧师](https://seasonloot.com/zh-cn/priest/) · [死亡骑士](https://seasonloot.com/zh-cn/deathknight/) · [萨满祭司](https://seasonloot.com/zh-cn/shaman/) · [法师](https://seasonloot.com/zh-cn/mage/) · [术士](https://seasonloot.com/zh-cn/warlock/) · [武僧](https://seasonloot.com/zh-cn/monk/) · [德鲁伊](https://seasonloot.com/zh-cn/druid/) · [恶魔猎手](https://seasonloot.com/zh-cn/demonhunter/) · [唤魔师](https://seasonloot.com/zh-cn/evoker/)

每个页面列出该职业**所有可装备的物品**，可以按部位或按来源浏览，还可以进一步按专精筛选 —— 比如武僧的酒仙 / 踏风 / 织雾。

## 你能查到什么

### 所有装备部位

头部、颈部、肩部、披风、胸部、腕部、手部、腰部、腿部、脚部、戒指、饰品、武器 —— 所有部位的装备汇集在一个页面。

### 所有掉落来源

清楚地看到每件装备的掉落出处：

- **11 个地下城** — 三杰殿、天际哨塔、密萨拉洞穴、风行者塔、萨隆矿坑、魔导师平台、节点 · 希纳斯、奥杰塔学院、梦境裂隙、虚空尖塔、奎尔丹纳斯进军
- **团本首领** — 当前团本的完整首领掉落列表
- **职业套装** — 每个职业的全部 9 件套装，包括各专精的 2 件套和 4 件套效果

### 每件装备显示什么

SeasonLoot 上的每件装备都会展示：

- **装备名称** — 暴雪官方本地化名称，和游戏内完全一致
- **装备部位** — 头部、胸部、武器、饰品等
- **副属性及精确数值** — 例如：暴击 82 / 全能 48
- **护甲类型** — 布甲、皮甲、锁甲、板甲
- **装备等级** — 地下城掉落 (装等 246)、团本掉落 (装等 269–276)、套装 (装等 263)
- **掉落来源** — 具体的地下城、团本首领或套装
- **武器子类型** — 法杖、长柄武器、单手剑、拳套、单手锤、单手斧、副手等
- **饰品效果** — 标注使用型（主动触发）或装备型（被动触发），以及主属性

## 强大的筛选功能

不用在上百件装备里慢慢翻。用内置筛选器精确找到你要的：

| 筛选项 | 功能 |
|--------|------|
| **专精** | 只显示你的专精能使用的装备（例如：神圣骑士、狂怒战士） |
| **属性** | 按副属性筛选 — 暴击、急速、精通、全能 |
| **来源** | 只看地下城掉落、团本掉落或套装 |
| **部位** | 直接跳转到某个装备部位 |
| **副本** | 筛选某个特定地下城的掉落 |
| **视图** | 在"按部位"和"按来源"两种浏览方式间切换 |
| **搜索** | 输入装备名称即时查找 |

**举个例子：** 你在玩一个急速/精通流的踏风武僧？选择踏风专精，勾选急速和精通，马上就能看到所有符合条件的装备 —— 以及你需要刷哪个地下城或打哪个团本首领。

## 数据来源与准确性

SeasonLoot 的所有数据**直接从魔兽世界游戏客户端中采集**。通过自研游戏内插件，读取冒险者手册中的地下城和团本掉落列表，以及套装展示系统，捕获每件装备的属性、装等和掉落来源。

各语言的装备名称来自**暴雪官方 Battle.net API**，确保装备名称、属性标签和提示信息与游戏内显示完全一致。

当前数据覆盖至暗之夜赛季1（补丁 4.2.x）的完整掉落池：

- **2,000+ 件装备**，覆盖全部 13 个职业
- **11 个大秘境赛季地下城**，含完整的首领掉落列表
- **3 个团本副本**，含英雄难度全部首领掉落
- **117 件职业套装**，覆盖 13 个职业，含各专精的套装效果
- **精确的装备等级** — 地下城 装等246、团本 装等269–276、套装 装等263
- **精确的副属性数值**

暴雪更新装备数据或新赛季上线时，SeasonLoot 会同步更新。

## SeasonLoot 适合谁？

- **刷本玩家** — 进本之前就知道该刷哪个地下城、打哪个首领
- **团本玩家** — 开团前查好哪些首领掉你的毕业装 (BiS)
- **练小号** — 快速了解一个不熟悉的职业有什么装备可以拿
- **配装规划** — 按照你想要的属性优先级筛选，规划最优装备方案
- **休闲玩家** — 轻松浏览当前赛季装备，不用在复杂的数据库网站里迷路
- **国际玩家** — 用你自己的母语查装备，所有名称都是暴雪官方翻译
