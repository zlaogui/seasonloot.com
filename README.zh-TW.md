## SeasonLoot 是什麼？

SeasonLoot 是一個免費的魔獸世界裝備掉落查詢網站，涵蓋 **魔獸世界：至暗之夜 賽季1** 的全部掉落資料。選擇你的職業，依照你想要的屬性進行篩選，就能立刻看到這個賽季所有你能穿的裝備 —— 在哪掉落、有什麼屬性、裝等多少。

不用裝插件，不用註冊帳號。打開 [seasonloot.com](https://seasonloot.com/zh-tw/) 就能開始規劃你的裝備。

## 涵蓋全部 13 個職業

每個職業都有專屬的掉落頁面，展示當前賽季的完整裝備列表：

[戰士](https://seasonloot.com/zh-tw/warrior/) · [聖騎士](https://seasonloot.com/zh-tw/paladin/) · [獵人](https://seasonloot.com/zh-tw/hunter/) · [盜賊](https://seasonloot.com/zh-tw/rogue/) · [牧師](https://seasonloot.com/zh-tw/priest/) · [死亡騎士](https://seasonloot.com/zh-tw/deathknight/) · [薩滿](https://seasonloot.com/zh-tw/shaman/) · [法師](https://seasonloot.com/zh-tw/mage/) · [乕術師](https://seasonloot.com/zh-tw/warlock/) · [武僧](https://seasonloot.com/zh-tw/monk/) · [德魯伊](https://seasonloot.com/zh-tw/druid/) · [惡魔獵人](https://seasonloot.com/zh-tw/demonhunter/) · [喚能師](https://seasonloot.com/zh-tw/evoker/)

每個頁面列出該職業**所有可裝備的物品**，可以按部位或按來源瀏覽，還可以進一步按專精篩選 —— 例如武僧的釀酒 / 御風 / 織霧。

### 所有掉落來源

清楚地看到每件裝備的掉落出處：

- **11 個地城** — 三杰殿、天際哨塔、密薩拉洞穴、乕風行者之塔、乕薩倫礦坑、乕乕魔導師平台、乕節點 · 希納斯、乕奧乕傑塔學院、乕夢境裂隙、虛空尖塔、乕奎爾丹納斯進軍
- **團隊副本首領** — 當前團隊副本的完整首領掉落列表
- **職業套裝** — 每個職業的全部 9 件套裝，包括各專精的 2 件套和 4 件套效果

### 每件裝備顯示什麼

SeasonLoot 上的每件裝備都會展示：

- **裝備名稱** — 暴雪官方本地化名稱，和遊戲內完全一致
- **裝備部位** — 頭部、胸部、武器、飾品等
- **副屬性及精確數值** — 例如：爆擊 82 / 臨機應變 48
- **護甲類型** — 布甲、皮甲、鎖甲、鎧甲
- **裝備等級** — 地城掉落 (裝等 246)、團隊副本掉落 (裝等 269–276)、套裝 (裝等 263)
- **掉落來源** — 具體的地城、團隊副本首領或套裝
- **武器子類型** — 法杖、長柄武器、單手劍、拳套、單手錘、單手斧、副手等
- **飾品效果** — 標註使用型（主動觸發）或裝備型（被動觸發），以及主屬性

## 強大的篩選功能

不用在上百件裝備裡慢慢翻。用內建篩選器精確找到你要的：

| 篩選項 | 功能 |
|--------|------|
| **專精** | 只顯示你的專精能使用的裝備（例如：神聖騎士、狂怒戰士） |
| **屬性** | 按副屬性篩選 — 爆擊、加速、精通、臨機應變 |
| **來源** | 只看地城掉落、團隊副本掉落或套裝 |
| **部位** | 直接跳轉到某個裝備部位 |
| **副本** | 篩選某個特定地城的掉落 |
| **視圖** | 在「按部位」和「按來源」兩種瀏覽方式間切換 |
| **搜尋** | 輸入裝備名稱即時查找 |

## 資料來源與準確性

SeasonLoot 的所有資料**直接從魔獸世界遊戲用戶端中採集**。透過自研遊戲內插件，讀取冒險者手冊中的地城和團隊副本掉落列表，以及套裝展示系統，擷取每件裝備的屬性、裝等和掉落來源。

各語言的裝備名稱來自**暴雪官方 Battle.net API**，確保裝備名稱、屬性標籤和提示資訊與遊戲內顯示完全一致。

當前資料涵蓋至暗之夜賽季1（補丁 4.2.x）的完整掉落池：

- **2,000+ 件裝備**，涵蓋全部 13 個職業
- **11 個傳奇鑰石賽季地城**，含完整的首領掉落列表
- **3 個團隊副本**，含英雄難度全部首領掉落
- **117 件職業套裝**，涵蓋 13 個職業，含各專精的套裝效果
- **精確的裝備等級** — 地城 裝等246、團隊副本 裝等269–276、套裝 裝等263
- **精確的副屬性數值**

暴雪更新裝備資料或新賽季上線時，SeasonLoot 會同步更新。

- **刷本玩家** — 進本之前就知道該刷哪個地城、打哪個首領

- **國際玩家** — 用你自己的母語查裝備，所有名稱都是暴雪官方翻譯
