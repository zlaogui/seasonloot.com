## SeasonLoot이란?

SeasonLoot은 **월드 오브 워크래프트: 내부의 전쟁 — 미드나이트 시즌 1**의 무료 전리품 테이블 브라우저입니다. 직업을 선택하고 원하는 스탯으로 필터링하면, 이번 시즌에 획득 가능한 모든 장비를 즉시 확인할 수 있습니다 — 어디서 드롭되는지, 어떤 스탯이 붙어 있는지, 아이템 레벨은 얼마인지.

애드온 설치 불필요. 계정 생성 불필요. [seasonloot.com](https://seasonloot.com/ko/)을 열고 바로 장비를 계획하세요.

## 13개 전 직업 지원

모든 WoW 직업마다 현재 시즌의 전체 장비 목록을 보여주는 전용 페이지가 있습니다:

[전사](https://seasonloot.com/ko/warrior/) · [성기사](https://seasonloot.com/ko/paladin/) · [사냥꾼](https://seasonloot.com/ko/hunter/) · [도적](https://seasonloot.com/ko/rogue/) · [사제](https://seasonloot.com/ko/priest/) · [죽음의 기사](https://seasonloot.com/ko/deathknight/) · [주술사](https://seasonloot.com/ko/shaman/) · [마법사](https://seasonloot.com/ko/mage/) · [흑마법사](https://seasonloot.com/ko/warlock/) · [수도사](https://seasonloot.com/ko/monk/) · [드루이드](https://seasonloot.com/ko/druid/) · [악마 사냥꾼](https://seasonloot.com/ko/demonhunter/) · [기원사](https://seasonloot.com/ko/evoker/)

각 페이지에는 해당 직업이 **장비할 수 있는 모든 아이템**이 나열됩니다. 부위별 또는 출처별로 탐색할 수 있으며, 특성별로 추가 필터링도 가능합니다 — 예: 수도사의 양조 / 풍운 / 운무.

## 무엇을 찾을 수 있나요?

### 모든 장비 부위

머리, 목, 어깨, 망토, 가슴, 손목, 손, 허리, 다리, 발, 반지, 장신구, 무기 — 모든 장비 부위가 한 페이지에.

### 모든 드롭 출처

각 아이템이 정확히 어디서 드롭되는지 확인하세요:

- **11개 던전** — 삼두정의 자리, 하늘빛 첨탑, 미스사라 동굴, 바람순찰자의 탑, 사론의 구덩이, 마법학자의 정원, 노드 시나스, 알게타르 아카데미, 꿈의 균열, 공허 첨탑, 쿠엘다나스 진격
- **레이드 보스** — 현재 티어의 전체 레이드 드롭 목록
- **세트 장비** — 모든 직업의 9세트 아이템, 특성별 2세트/4세트 효과 포함

### 각 아이템에 표시되는 정보

SeasonLoot의 모든 아이템에는 다음이 표시됩니다:

- **아이템 이름** — 블리자드 공식 현지화 이름, 게임 내와 동일
- **장비 부위** — 머리, 가슴, 무기, 장신구 등
- **부가 스탯 및 정확한 수치** — 예: 치명타 82 / 유연성 48
- **방어구 유형** — 천, 가죽, 사슬, 판금
- **아이템 레벨** — 던전 드롭 (ilvl 246), 레이드 드롭 (ilvl 269–276), 세트 (ilvl 263)
- **드롭 출처** — 특정 던전, 레이드 보스 또는 세트
- **무기 세부 유형** — 지팡이, 장창, 한손 검, 권투 무기, 한손 둔기, 한손 도끼, 보조 무기 등
- **장신구 효과** — 사용형(활성) 또는 장착형(수동) 표시, 주 스탯 포함

## 강력한 필터

수백 개의 아이템을 스크롤할 필요 없습니다. 내장 필터로 정확히 원하는 것을 찾으세요:

| 필터 | 기능 |
|------|------|
| **특성** | 해당 특성이 장비할 수 있는 아이템만 표시 (예: 신성 성기사, 분노 전사) |
| **스탯** | 부가 스탯별 필터 — 치명타, 가속, 특화, 유연성 |
| **출처** | 던전 드롭, 레이드 드롭, 세트만 표시 |
| **부위** | 특정 장비 부위로 바로 이동 |
| **던전** | 특정 던전의 드롭만 필터링 |
| **보기 모드** | 부위별 또는 출처별 보기 전환 |
| **검색** | 아이템 이름을 입력하여 즉시 검색 |

**예시:** 가속/특화 풍운 수도사를 플레이 중이신가요? 풍운 특성을 선택하고 가속과 특화를 체크하면, 조건에 맞는 아이템만 즉시 확인 — 파밍해야 할 던전이나 레이드 보스도 함께 표시됩니다.

## 즐겨찾기 & 공유

- 아이템의 **하트 아이콘**을 클릭하여 즐겨찾기에 저장
- 즐겨찾기는 브라우저에 로컬 저장 — 다음 방문에도 유지, 로그인 불필요

## 데이터 출처 및 정확성

SeasonLoot의 모든 데이터는 **월드 오브 워크래프트 게임 클라이언트에서 직접 수집**됩니다. 자체 제작 게임 내 애드온을 통해 모험 안내서의 던전 및 레이드 드롭 목록과 세트 표시 시스템을 읽어, 모든 아이템의 스탯, 아이템 레벨, 드롭 출처를 캡처합니다.

각 언어의 아이템 이름은 **블리자드 공식 Battle.net API**에서 가져와, 아이템 이름, 스탯 라벨, 툴팁이 게임 내 표시와 정확히 일치합니다.

현재 데이터는 미드나이트 시즌 1(패치 4.2.x)의 전체 드롭 풀을 포함합니다:

- **2,000개 이상의 아이템**, 13개 전 직업 포함
- **11개 쐐기돌 시즌 던전**, 전체 보스 드롭 목록 포함
- **3개 레이드 인스턴스**, 영웅 난이도 전체 보스 드롭 포함
- **117개 세트 아이템**, 13개 직업별 특성 세트 효과 포함
- **정확한 아이템 레벨** — 던전 ilvl 246, 레이드 ilvl 269–276, 세트 ilvl 263
- **정확한 부가 스탯 수치**

블리자드가 아이템을 업데이트하거나 새 시즌이 출시되면 SeasonLoot도 함께 업데이트됩니다.

## SeasonLoot은 누구를 위한 것인가요?

- **던전 플레이어** — 입장 전에 어떤 던전을 파밍하고 어떤 보스를 노려야 하는지 파악
- **레이더** — 레이드 전에 BiS(Best in Slot) 장비를 드롭하는 보스 확인
- **부캐 육성** — 처음 장비를 맞추는 직업에 어떤 아이템이 있는지 빠르게 파악
- **빌드 계획** — 원하는 스탯 우선순위로 필터링하여 최적의 장비 세트 계획
- **캐주얼 플레이어** — 복잡한 데이터베이스 사이트 없이 현 시즌 장비를 쉽게 탐색
- **글로벌 플레이어** — 블리자드 공식 번역으로 모국어로 장비 검색

## 무료, 애드온 불필요, 어디서나 사용 가능

SeasonLoot은 브라우저에서 완전히 실행됩니다. 다운로드할 것도, 설치할 애드온도, 가입할 계정도 없습니다. 빠르게 로드되며 PC, 태블릿, 스마트폰 모두에서 작동합니다.
