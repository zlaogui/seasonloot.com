## Что такое SeasonLoot?

SeasonLoot — это бесплатный браузер таблиц добычи для **World of Warcraft: The War Within — Midnight Сезон 1**. Выберите свой класс, отфильтруйте по нужным характеристикам и мгновенно увидите каждый предмет экипировки, доступный в этом сезоне — где он выпадает, какие характеристики имеет и какой у него уровень предмета.

Не нужно устанавливать аддоны. Не нужно создавать аккаунт. Просто откройте [seasonloot.com](https://seasonloot.com/ru/) и начните планировать свою экипировку.

На каждой странице перечислены **все носимые предметы** для данного класса, организованные по ячейке или по источнику. Вы можете дополнительно фильтровать по специализации — например, Хмелевар / Танцующий с ветром / Ткач туманов для Монаха.

### Что показывает каждый предмет

Каждый предмет на SeasonLoot отображает:

- **Название предмета** — Официальное локализованное Blizzard название на вашем языке
- **Ячейка экипировки** — Голова, Грудь, Оружие, Аксессуар и т.д.
- **Вторичные характеристики с точными значениями** — напр., Крит 82 / Универс. 48
- **Тип брони** — Ткань, Кожа, Кольчуга или Латы
- **Уровень предмета** — Добыча из подземелий (ilvl 246), Добыча из рейдов (ilvl 269–276), Комплектные предметы (ilvl 263)
- **Источник добычи** — Конкретное подземелье, рейдовый босс или комплект
- **Подтип оружия** — Посох, Древковое оружие, Одноручный меч, Кистевое оружие, Одноручная булава, Одноручный топор, Левая рука и т.д.
- **Эффекты аксессуаров** — Является ли аксессуар активируемым (по использованию) или пассивным (при надевании), с указанием основной характеристики

## Мощные фильтры

Не пролистывайте сотни предметов. Используйте встроенные фильтры, чтобы найти именно то, что ищете:

| Фильтр | Что делает |
|--------|-----------|
| **Специализация** | Показать только предметы, доступные для вашей специализации (напр., Свет Паладин, Неистовство Воин) |
| **Характеристики** | Фильтр по вторичной характеристике — Крит, Скорость, Искусность или Универс. |
| **Источник** | Показать только добычу из Подземелий, Рейдов или Комплектные предметы |
| **Ячейка** | Перейти непосредственно к конкретной ячейке экипировки |
| **Подземелье** | Фильтр по добыче из конкретного подземелья |
| **Режим просмотра** | Переключение между просмотром по ячейке или по источнику |
| **Поиск** | Введите название предмета для мгновенного поиска |

**Пример:** Играете Монахом Танцующим с ветром со Скорость/Искусность? Выберите специализацию Танцующий с ветром, отметьте Скорость и Искусность, и мгновенно увидите только подходящие предметы — вместе с конкретным подземельем или рейдовым боссом, которого нужно фармить.

## Избранное и обмен

- Нажмите на **значок сердца** на любом предмете, чтобы сохранить его в избранное
- Ваше избранное сохраняется между посещениями — хранится локально в браузере, вход не требуется
- Используйте **«Спросить друга»**, чтобы создать ссылку для обмена списком экипировки и отправить её согильдийцам

## Источник данных и точность

Все данные на SeasonLoot **собраны непосредственно из игрового клиента World of Warcraft** с помощью специальных внутриигровых аддонов. Процесс сбора читает Путеводитель по приключениям (таблицы добычи подземелий и рейдов) и систему отображения комплектов, чтобы зафиксировать каждый предмет, его характеристики, уровень предмета и источник добычи.

Названия предметов на каждом языке получены из **официального API Battle.net от Blizzard**, что гарантирует точное соответствие названий предметов, меток характеристик и подсказок тому, что вы видите в игре.

Текущий набор данных охватывает полный пул добычи Midnight Сезон 1 (Патч 4.2.x):

- **2 000+ предметов** по всем 13 классам
- **8 сезонных подземелий Мифик+** с полными таблицами добычи боссов
- **3 рейдовых инстанса** с добычей всех боссов на Героической сложности
- **117 комплектных предметов** по 13 классам с бонусами комплектов по специализациям
- **Точные уровни предметов** — Подземелье ilvl 246, Рейд ilvl 269–276, Комплект ilvl 263
- **Точные значения вторичных характеристик** для каждого предмета

Данные обновляются при каждом изменении предметов Blizzard или запуске нового сезона.
